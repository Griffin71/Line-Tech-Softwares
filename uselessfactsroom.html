About the App

Useless Facts Room is a cross-platform mobile trivia application designed to entertain, challenge, and surprise users with interesting and completely useless facts. The app includes difficulty levels, achievements, sound effects, haptic feedback, and offline play support.

Multiple Difficulties

Easy, Medium, and Hard modes for all skill levels

Installation

Step-by-step instructions for Android and iOS

Android Installation

  1. Install Expo Go from Google Play Store.
  2. Open Expo Go.
  3. Scan the provided QR code or open the shared project link.
  4. The app will launch instantly.

iOS Installation

  1. Install Expo Go from the App Store.
  2. Open Expo Go.
  3. Scan the project QR code.
  4. App Store release pending due to developer account costs.

Developer Notes

Useless Facts Room is built with Expo and React Native. We use AsyncStorage for local persistence and plan continued updates to improve performance and add new content.

Need Help

  • App won't start: Ensure Expo Go is installed and scan the correct QR code.
  • Missing content/offline issues: Open the app while online to refresh cached data.
  • Crashes or freezes: Restart the app and check for updates; contact support if persistent.
